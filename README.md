Proyectero is a management web app implemented for py4web framework that allows to manage small projects. It is devised as a tool for students and teachers in technical education (IT); Developed at Escuela Técnica 29 DE 6 "Reconquista" (Buenos Aires - Argentina)

How to try this project

For detailed instructions, see [Admin manual] (https://github.com/alanetkinet29/proyectero/wiki/Admin-manual)

- First, you must have a development py4web environment running in your system
- Then, copy or unzip proyectero files to the [route to py4web]/apps/proyectero folder
- Run py4web for development server
- Open a browser and go to the http://localhost:8000/proyectero address
- To manage the app you can use the py4web built-in dashboard at http://localhost:8000/_dashboard

Note: py4web requires a Python interpreter to run

Also, see the [proyectero user's manual] (https://github.com/alanetkinet29/proyectero/wiki/User's-manual)
